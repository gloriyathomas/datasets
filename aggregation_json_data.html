So far in the course we've been using the PUT command to index each document separately into Elasticsearch. The example below is another way and is more efficient because we're sending many documents as part of the same request. This is known as the BULK API. Feel free to read more about it in the Elasticsearch reference documentation found at this link: https://www.elastic.co/guide/en/elasticsearch/reference/current/docs-bulk.html
